## Python Example

```python
import requests
from typing import List

# Replace with your actual API key
API_KEY = "your_api_key_here"

def create_room(
    character_ids: List[str],
    session: requests.sessions.Session
):
    response = session.post(
        url="https://api.genova.gg/v1/rooms",
        headers={"Content-Type": "application/json", "x-api-key": API_KEY},
        json={
            "character_ids": character_ids,
        },
    )
    if response.status_code == 200:
        response = response.json()
        return response["id"]
    return None
```

The example above demonstrates how to create a new room with specified character IDs. The function:
- Takes a list of character IDs and a session object
- Makes a POST request to the rooms endpoint
- Returns the room ID on success, None on failure
- Uses proper authentication via x-api-key header
