## Authentication & API Keys

All API endpoints are authenticated using `x-api-key` in your request header

```python
curl "https://api-parallel.aideate.ai/v5/<api-name>" \
  -H "Content-Type: application/json" \
  -H "x-api-key: $AIDEATE_API_KEY" \
  -d '{
    ...
  }'
```
